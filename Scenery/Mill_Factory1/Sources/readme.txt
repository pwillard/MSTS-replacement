== Mill Factory 1 

* This is a scenery object.
* This scenery object is a part of the Mill Factory 1 scenery object set. More to come later.
* The final `S` file and texture are renamed after export so the `S` file needs to be edited.
* An exported PSD file is provided for reference as well as the UVMAP file. (might be outdated)
* The master texture was created in about 30 minutes using Affinity Designer 2 exported as PNG and converted to DDS using Paint.net.

This folder is provided to show how easy this stuff really is.
